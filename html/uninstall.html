Automated Removal

  1. Go to the admin section of your WordPress site.
  2. Select the SSO menu item.
  3. Select the Advanced Settings sub-menu item.
  4. If you would like to remove WPSSO Core settings and all custom post, term, and user metadata, make sure the "Remove All Settings on Uninstall" option is checked.
  5. Select the Plugins menu item.
  6. Select the Installed Plugins sub-menu.
  7. Scroll down to the "WPSSO Core" plugin (or use the Search field), and click its Deactivate link.
  8. Scroll down to the "WPSSO Core" plugin (or use the Search field), and click its Delete link.
  9. Click on the Yes, Delete these files and data button.

Manual Removal

Note that removing the plugin folder manually will not remove its settings from the database.

  1. Remove the wordpress/wp-content/plugins/wpsso/ folder from your website.
